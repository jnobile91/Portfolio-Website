My personal Portfolio Website

HTML, CSS, JavaScript

This website was created using a combination of HTML, CSS, and JavaScript techniques learned from online materials. I have a small amount of CSS and JavaScript experience from my PHP project, but there is a plethora of documentation and information available on the internet to learn from. This project appeared incredibly challenging with minimal experience in front-end development.

After completing this website I have learned so much about the potential customization for front-end development using CSS and basic JavaScript functions. By organizing my HTML items into versatile classes I was able to universally change the settings of similar components throughout the page. I also implemented some JavaScript functionality in order to have the navigation menu open as expected, having the hamburger menu icon transform into an X to indicate that another click will close the menu. There were several other CSS techniques used throughout the website, utilizing transform commands in order to have interactive components via the "My Work" button and portfolio items.

This project was particularly exciting for me. I consider myself a creative person. Finding a way to utilize those skills and translate it on to a web page was extremely satisfying. Despite a lack of front-end experience, I consistently found myself wanting to continue improving the website. Throughout the day I would catch myself thinking about features to research and implement into the website. It also fulfilled a need by providing a publicly accessible platform for potential employers to view my work. Overall, I am extremely satisfied with the result, but am constantly looking for new additions to include.
