Basic PHP Database Application

HTML, PHP, SQL, JavaScript

This project was the first time I constructed a user-facing interface to interact with a database using SQL. It consists of techniques required to create, add, and delete features from a database. I also included user validation techniques in order to require user registration before interacting with the database.

Techniques that were implemented include user validation and password hashing. Using validation methods, users are required to register with the platform before accessing the database. Passwords are stored using a SHA-256 hashing technique, then adding "salt" to the beginning and end of each hashed password. The "salt" is randomly generated via a custom function that creates a string of random characters. Once a user account is created, the login credentials are validated using custom functions and JavaScript validation. This project also utilizes session management, ensuring that the user has a seamless connection each time they are accessing the application, complete with proper logout functionality. List, Search, Add, and Delete functionality utilizes placeholder queries to properly interact with the database. If entering a new entry the input is ran through several validation checks to ensure data integrity.

I found this project particularly challenging since this was the first time I was creating a thorough application from scratch. Researching the proper functionality of user registration/validation and session management proved to be more involved than my previous programming applications, but was incredibly rewarding to finally get it working. Most applications these days require user registration, so learning these techniques was exceptionally valuable. The user functionality is something that I plan to continue to research and implement into future projects.
